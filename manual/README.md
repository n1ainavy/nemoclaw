# Manual Uploads

Place PDF files in this folder for consolidation.

## Instructions

1. Drop any PDF policy documents into this folder
2. Use subfolders to organize by topic (optional)
3. On next script run, all PDFs here will be:
   - Validated for content
   - Consolidated into `manual_consolidated.pdf`
   - Mapped in `manual_manifest.md`
   - Copied to the `consolidated/` quick-access folder

## Subfolder suggestions

```
manual/
├── local-instructions/    # Unit-specific policies
├── command-notices/       # Command-level guidance
├── reference/             # Useful reference docs
└── other/                 # Miscellaneous
```
